# Practice Apps

Swift / SwiftUI の学習用に作った練習アプリをまとめています。

##  アプリ一覧

| アプリ名       | 説明                                   | 技術              |
|----------------|----------------------------------------|-------------------|
| おみくじ       | ランダムに運勢を表示するアプリ         | SwiftUI           |
| カウント       | ボタンで数字が増える基本のカウント     | SwiftUI           |
| クイズ         | 問題を出して正解・不正解を表示         | SwiftUI / MVVM    |
| タイマー       | カウントダウンする簡単なタイマー       | SwiftUI           |
| ToDoリスト     | チェック機能付きの買い物リスト風       | SwiftUI / List    |
